# /ghj-a11y

GroupHomeJuju のアクセシビリティチェックを実行します。

## 実行内容
1. **accessibility-specialist エージェント起動**: WCAG 2.1 AA準拠チェック
2. **自動テスト実行**: axe-core による検証
3. **手動チェック項目**: キーボード操作・スクリーンリーダー対応
4. **高齢者・障害者対応**: 特別配慮事項の確認
5. **改善提案**: 具体的な修正方法の提示
6. **レポート作成**: アクセシビリティ状況の文書化

**引数**: $ARGUMENTS (チェックしたいページ・機能)

**使用例**:
- `/ghj-a11y index.html`
- `/ghj-a11y お問い合わせフォーム`
- `/ghj-a11y 全ページ`

障害者グループホームサイトとして模範的な実装を目指します。
